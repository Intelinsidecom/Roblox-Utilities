# Roblox-Utilities
A utility for automating, simplifying the process for specific checks or uses regarding Roblox Clients. Written in C#.

## Utility Features:
1. Private, Public and Encrypted signature key generation for signing scripts
2. Script signing with generated key above with rbxsig signature

## Potential features to add
1. GUI app version
2. Universal FFLag extraction
3. MD5 client hashing

## How to compile, use?
In order to compile you need atleast .NET Framework 4.8 sdk or Visual Studio 2015.

Using is quite simple, the Console App already comes with an interactive menu, just run the app and enjoy.
Console App also supports arguments for automatization purposes, to see a list of them, pass --help within the app in command line.
