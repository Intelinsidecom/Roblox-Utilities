# Roblox-Utilities
A utility for automating, simplifying the process for specific checks or uses regarding Roblox Clients. Written in C#.

## Utility Features:
1. Private, Public and Encrypted signature key generation for signing scripts
2. Script signing with generated key above with rbxsig signature

## Potential features to add
1. GUI app version
2. Universal FFLag extraction
3. MD5 client hashing
